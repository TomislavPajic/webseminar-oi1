SLIČNE PLATFORME

Java je ostvarila veliki uspjeh sa svojim ciljem da se kod napisan jednom pokreće na bilo kojoj platformi, pa je bilo i pokušaja sličnih koncepata. Najuvjerljiviji je Microsoft-ov .NET Framework. Pojavio se 2002. godine te je napravljen na jako sličnom konceptu. Glavne razlike su da se može koristiti samo na Windows platformi dok se Java može koristiti svugdje. Prednost nad Java-om je to što je napravljen sa namjerom da podržava sve programske jezike dok je Java platforma u početku napravljena s namjerom da podržava samo Java programski jezik.
