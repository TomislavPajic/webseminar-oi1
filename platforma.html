JAVA KAO SOFTVERSKA PLATFORMA

Java nam pruža sistem za razvijanje aplikacijskog softvera i njegovo puštanje u kompjutersko okruženje između različitih platformi. Java se koristi na vrlo različitim računalnim platformama, od ugrađenih i mobilnih uređaja do serverskih farmi i super-računala.

Kod koji koristimo na Java-inoj platformi se piše u njenom programskom jeziku. Ona je dizajnirana kao set programa na kojima koristimo i programiramo kod u Java programskom jeziku. Uključuje tzv. virtual machine, kompajler i set biblioteka.

Platforma se sastoji od nekoliko programa, svaki od kojih pridonosi nekoliko od njenih ukupnih mogućnosti. Kompajler koji izvorni kod pretvara u tzv. bytecode koji se dalje obrađuje dolazi kao dio Java Development Kit-a. Nadalje, Java Runtime Environment uzima bytecode i pretvara ga u izvorni kod stroja.

Dijelimo je na 4 implementacije:

  • Java Card – omogućuje malim aplikacijama da se sigurno pokrenu na uređajima sa malo memorije i slabim procesorom
  • Java Micro Edition – set specifičnih profila za uređaje sa slabom memorijom i procesorskom snagom pa se najčešće koristi na mobilnim uređajima, printerima i sl.
  • Java Standard Edition – standardna verzija, koristi se za razvoj računalnih aplikacija, serverskih aplikacija i sl.
  • Java Enterprise Edition – osnovano na bazi standardne verzije ali uz mnoge dodatke za razvoj aplikacija s nekoliko razina funkcionalnosti na relaciji klijent-server.
