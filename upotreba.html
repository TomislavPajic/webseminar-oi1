UPOTREBA

Java ima puno različitih uporaba, i koristi se na vrlo različitim mjestima. Utjelovljenje cilja prenosivosti su upravo aplikacije za mobilni operativni sustav Android. Iako je on napisan većinom u C-u, aplikacije za Android napisane su na osnovi Java-e.

Osim mobilnih aplikacija, Java je oduvijek bila velika i na internetu. U zadnje vrijeme možemo bez nje jer dolaze neke nove tehnologije, ali 1990-ih je bila jako raširena zbog Applet-a. Danas se koristi najviše u sferi e-trgovine ali i mnoge ostale web aplikacije su napisane u Java-i.

Java je raširena i u financijskoj industriji. Svjetske investicijske banke koriste je najčešće za uredske elektronske sustave trgovanja ali koristi se i za sustave potvrde, projekte procesiranja podataka i mnoge druge. Najčešće se koristi za pisanje serverskih aplikacija.

Osim što je programski jezik, Java je zapravo i set računalnih softverskih paketa. Oni korisnicima omogućuju pokretanje aplikacija napisanih na Java-i. Java kao platforma također nije vezana za jedan operativni sustav nego je prilagodljiva.

Uz ovako raširenu upotrebu Java-e, u pitanje dolazi njena sigurnost. Oracle često izbacuje nove verzije i nadogradnje njihovih aplikacija ali one većinom ne donose neke značajne promjene već samo zakrpe sigurnosnih propusta. Primjer ovakvih zakrpi bio je Java October 2013 Critical Patch Update kada je zakrpano čak 12 kritičnih propusta zbog kojih su hakeri mogli dobiti pristup cijelome operativnom sustavu napadnutog korisnika.
