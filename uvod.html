UVOD

Programski jezik Java je dizajniran kao moderni programski jezik za opću namjenu. Baziran je na objektima pa pripada u skupinu objektno-orijentiranih programskih jezika. Java programerima dozvoljava korištenje njihovog kompajliranog koda na bilo kojoj platformi koja podržava Java-u bez potrebe za preinakama. Trenutno je jedan od najpopularnijih programskih jezika prema TIOBE indeksu.

Prva verzija Java-e objavljena je 1995. godine, razvijena od strane Jamesa Goslinga i grupe manje poznatih inženjera iz tvrtke Sun Microsystems koja je danas u vlasništvu Oracle Corporation. Bila je jezgrena komponenta njihove Java platforme. Razvoj je počeo modifikacijom programskog jezika UCSD Pascal. Nakon toga je oblikovan tako da je sintaksa slična jezicima C i C++. Prve verzije ovog jezika zvale su se Oak, ali je kasnije preimenovan u Java.

Najveća prednost u odnosu na ostatak dotadašnjih programskih jezika bila je mogućnost izvođenja programa pisanih u Java-i na svim operativnim sustavima za koje postoji JVM ( Java Virtual Machine ) bez potrebe za prilagođavanjem poput jezika C.

Java je dinamičan programski jezik, a to znači da ima mogućnost učitavanja kod programa s diska ili preko mreže i izvršavati ga za vrijeme rada. To je iskorišteno u distribuiranim sustavima u Javi.

Pri stvaranju Java programskog jezika, bilo je pet glavnih ciljeva:

  • mora biti „jednostavan, objektno-orijentiran, i poznat“
  • mora biti „robustan i siguran“
  • mora biti „arhitekturno neutralan i prenosiv“
  • mora se izvršavati sa „visokim performansama“
  • mora biti „interpretiran, navojen i dinamičan“
