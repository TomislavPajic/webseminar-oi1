POVIJEST

Razvoj Java-e započeli su inženjeri James Gosling, Mike Sheridan i Patrick Naughton iz tvrtke Sun Microsystems u lipnju 1995. godine. Na početku je dizajnirana za interaktivnu televiziju ali je bila daleko prenapredna za digitalnu kabelsku televiziju tog vremena.

Projekt se na početku zvao Oak, promijenio je ime u Green i na kraju dobio finalnu oznaku Java. Gosling je dizajnirao Java-u sa sintaksom jezika C i C++ kako bi je sistemski i aplikacijski programeri brzo shvatili.

Java je bila prilično sigurna i imala je mogućnosti konfiguracije sigurnosnih postavki. Omogućila je ograničenja pristupa mreži i datotekama. Veliki web preglednici ubrzo su uključili mogućnost pokretanja Java koda unutar svojih web stranica pridonoseći njenoj brzoj popularizaciji.

Sun Microsystems je većinu svojih Java implementacija ostavio dostupne bez naknade. Generirali su prihod od Java-e putem prodaje licenci za specijalizirane proizvode poput Java Enterprise System-a.

Oracle Corporation preuzima Sun Microsystems u razdoblju od 2009. do 2010. godine te se predstavlja kao „upravitelj Java tehnologije s neumornom predanošću za poticanje zajednice sudjelovanja i transparentnosti“.

Danas je Java jedan od najpopularnijih programskih jezika i koristi se na svim platformama, od prijenosnih računala do velikih podatkovnih centara.

Glavni dizajner i osnivač ovog programskog jezika, James Gosling 2. travnja 2016. napušta Oracle Corporation.
